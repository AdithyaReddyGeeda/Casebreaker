# ⚡ Automated Workflow Setup - JUST DO THIS

You're tired. Here's the EXACT steps. Just follow them.

---

## **Step 1: Get Your API Keys (10 minutes)**

### **A. Tripo API**
```
1. Go to: https://www.tripo3d.ai/api
2. Click "Get API Access"
3. Fill form
4. Wait for approval (usually instant)
5. Copy your API key
```

### **B. ElevenLabs TTS**
```
1. Go to: https://elevenlabs.io
2. Sign up (free)
3. Go to: https://elevenlabs.io/voice-lab
4. Pick a voice (or use "Bella" by default)
5. Go to settings, copy your API key
```

### **C. Deepgram**
```
1. Go to: https://console.deepgram.com
2. Sign up (free)
3. Create API key (instant)
4. Copy it
```

---

## **Step 2: Add Keys to .env.local (2 minutes)**

Create/edit `.env.local` in your project root:

```
NEXT_PUBLIC_TRIPO_API_KEY=your_tripo_key_here
NEXT_PUBLIC_ELEVENLABS_API_KEY=your_elevenlabs_key_here
NEXT_PUBLIC_ELEVENLABS_VOICE_ID=bella
NEXT_PUBLIC_DEEPGRAM_API_KEY=your_deepgram_key_here
```

Done.

---

## **Step 3: Initialize the Service (3 minutes)**

In your app's main entry point (e.g., `_app.tsx` or main `layout.tsx`):

```typescript
import { initializeMysteryService } from "@/game/lib/mystery/MysteryToGameAssetsService";

// Run once at startup
useEffect(() => {
  initializeMysteryService({
    tripoApiKey: process.env.NEXT_PUBLIC_TRIPO_API_KEY!,
    elevenLabsApiKey: process.env.NEXT_PUBLIC_ELEVENLABS_API_KEY!,
    elevenLabsVoiceId: process.env.NEXT_PUBLIC_ELEVENLABS_VOICE_ID!,
    deepgramApiKey: process.env.NEXT_PUBLIC_DEEPGRAM_API_KEY!,
  });
}, []);
```

Done.
